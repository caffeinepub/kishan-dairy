# Specification

## Summary
**Goal:** Enable instant delivery indicator for all dairy products in the catalog.

**Planned changes:**
- Add `isInstant` boolean field to Product type in backend
- Update all existing products (milk, curd, paneer, butter, ghee) to enable instant delivery
- Display "Instant Delivery" badge on product cards for products with instant delivery enabled

**User-visible outcome:** Users can see which products are available for instant delivery through a prominent badge displayed on each product card.
